If a local variable is declared but not used, it is dead code and should be removed. Doing so will improve maintainability as developers will not wonder what the variable is used for.

Non-compliant Code Example

_method exemplar.hello(name)
  _local message << "Hello " + name  # message is an unused variable
  print("Hello " + name)
_endmethod

Compliant Code Example

_method exemplar.hello(name)
  _local message << "Hello " + name
  print(message)
_endmethod

If you have enabled checking the parameters as well, you should use all parameters as well.

_method exemplar.hello(name) # name is an unused parameter
  _local message << "Hello World!"
  print(message)
_endmethod

Compliant Code Example

_method exemplar.hello(name)
  _local message << "Hello " + name
  print(message)
_endmethod
